Objetivo

Receber uma mensagem de lead, analisá-la com IA e salvar os dados classificados automaticamente em uma planilha — sem intervenção humana.

Arquitetura

Fluxo completo —
nó por nó

1
Webhook
n8n-nodes-base.webhook · Método POST
Recebe a mensagem do lead. Path: zyllah-lead — URL de produção: http://localhost:5678/webhook/zyllah-lead
Recebe os campos: nome, telefone, mensagem
2
HTTP Request — API Claude
n8n-nodes-base.httpRequest · POST → api.anthropic.com
Envia a mensagem para a API Claude (Haiku) com prompt de classificação. Retorna JSON com classificacao (alto/medio/baixo) e resposta_sugerida.
Headers:
  x-api-key: SUA_CHAVE_SK-ANT
  anthropic-version: 2023-06-01
  content-type: application/json

Body:
  model: claude-haiku-4-5-20251001
  max_tokens: 1024
  messages[0].content:
    "Analise a mensagem abaixo e retorne APENAS um JSON
     com 'classificacao' (alto/medio/baixo) e
     'resposta_sugerida'. Mensagem: {{ $('Webhook').item.json.body.mensagem }}"
3
Edit Fields (Set)
n8n-nodes-base.set · Todos os campos como String
Processa e limpa os dados antes de gravar no Sheets. Extrai os valores do JSON retornado pela API com .replace() + .match().
⚠ Obrigatório: sempre usar o nó Set antes do Sheets. Expressões com funções longas aplicadas diretamente no Sheets causam o erro de = sendo interpretado como fórmula.
CampoExpressão n8n
nome{{ $('Webhook').item.json.body.nome }}
telefone{{ $('Webhook').item.json.body.telefone }}
mensagem{{ $('Webhook').item.json.body.mensagem }}
classificacao.match(/"classificacao":\s*"([^"]+)"/)[1]
resposta_sugerida.match(/"resposta_sugerida":\s*"([^"]+)"/)[1]
4
Google Sheets — Append Row
n8n-nodes-base.googleSheets · Operação: append · Mapping: defineBelow
Grava a linha no Sheets usando referências simples ={{ $json.campo }} — nunca expressões longas diretamente aqui.

Replicação

Como replicar para
um novo cliente

Sete passos para ter o sistema funcionando para um novo cliente sem criar nada do zero.

1
Criar planilha no Google Sheets Colunas: nome, telefone, mensagem, classificacao, resposta_sugerida
2
Duplicar o workflow no n8n Botão "Duplicate" no menu do workflow — nunca editar o original
3
Atualizar o path do webhook Renomear para o nome do cliente — ex: cliente-dra-ana
4
Atualizar o prompt no HTTP Request Substituir o contexto genérico pelo contexto específico do cliente (especialidade, nome, tom de voz)
5
Reconectar o Google Sheets Apontar para a planilha do novo cliente
6
Publicar o workflow Toggle "Active" no n8n — sem isso o webhook não responde em produção
7
Testar com curl Usar a nova URL de produção com dados fictícios do cliente
