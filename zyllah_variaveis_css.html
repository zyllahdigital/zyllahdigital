Bloco completo — copiar e colar

O :root oficial

Este é o único bloco necessário. Substitui qualquer `:root` anterior. Inclui todas as variáveis de cor, texto e contraste validadas conforme o brandbook seção 09.

CSS — colar no <style> de qualquer documento 
:root {
  /* ── CORES BASE ── */
  --ink:          #1A1714;   /* fundo principal */
  --ink-soft:     #3D3830;   /* texto corpo sobre claro */
  --cream:        #FAF7F2;   /* fundo secundário */
  --cream-mid:    #F0EBE1;   /* fundo terciário */
  --gold:         #B8976A;   /* destaque principal */
  --gold-light:   #D4B896;   /* itálicos, hover */
  --gold-dark:    #8A6A3E;   /* labels sobre claro */
  --gold-pale:    #F5EFE6;   /* fundos suaves */
  --rose:         #C47A7A;   /* alertas, problemas */
  --rose-light:   #F5EDED;   /* fundo rose */
  --sage:         #7A9B8A;   /* confirmações, ok */
  --sage-light:   #EDF5F2;   /* fundo sage */

  /* ── TEXTO SOBRE FUNDO ESCURO (ink) ── */
  --t-escuro-titulo:      rgba(255,255,255,1.0);   /* títulos — nunca abaixo de 0.85 */
  --t-escuro-corpo:       rgba(255,255,255,0.75);  /* parágrafos — nunca abaixo de 0.65 */
  --t-escuro-secundario:  rgba(255,255,255,0.55);  /* labels, badges — nunca abaixo de 0.45 */
  --t-escuro-ornamental:  rgba(255,255,255,0.20);  /* decorativo apenas — não usar em texto lido */

  /* ── TEXTO SOBRE FUNDO CLARO (cream) ── */
  --t-claro-titulo:       #1A1714;                 /* títulos — sempre sólido */
  --t-claro-corpo:        #3D3830;                 /* parágrafos — nunca mais claro que #5A5248 */
  --t-claro-secundario:   rgba(61,56,48,0.65);     /* labels, badges — nunca abaixo de 0.55 */
  --t-claro-ornamental:   rgba(61,56,48,0.20);     /* decorativo apenas — não usar em texto lido */

  /* ── BORDAS E DIVISORES ── */
  --borda-gold:           rgba(184,151,106,0.20);  /* divisores sutis */
  --borda-gold-forte:     rgba(184,151,106,0.40);  /* bordas de card */
  --borda-escura:         rgba(255,255,255,0.06);  /* divisores sobre ink */
}
Referência rápida — quando usar cada variável

Guia de uso
por contexto

A regra é simples: se o usuário precisa ler, use as variáveis de texto. Se é apenas decoração, as ornamentais são permitidas.

Atenção: a maioria dos problemas de contraste nos documentos atuais vem do uso de rgba(255,255,255,0.2) ou rgba(255,255,255,0.3) em rodapés e labels que deveriam ser lidos — não decorativos. Ao revisar um documento, procure por valores de opacidade abaixo de 0.45 em texto funcional e substitua pelas variáveis corretas acima.

Como aplicar na revisão

Checklist de
revisão rápida

Ao abrir qualquer documento HTML da Zyllah para edição, percorra esses quatro pontos antes de fazer qualquer outra mudança.

1

Substituir o :root antigo

Apagar o :root existente e colar o bloco oficial deste documento. Isso resolve a maioria dos problemas de uma vez.

2

Buscar por opacidade baixa em texto

No editor, buscar 0.2), 0.25), 0.3). Para cada resultado: é texto lido? Substituir pela variável correta. É ornamental? Manter.

3

Verificar rodapés e badges

São os lugares onde o problema aparece mais. Rodapé sobre fundo escuro: mínimo var(--t-escuro-secundario). Nunca ornamental em texto de rodapé.

4

Testar no celular

Telas de celular têm brilho variável. Um texto legível no monitor pode ser ilegível ao ar livre. Se parecer difícil de ler no celular em ambiente claro, aumentar a opacidade um nível.
